<COORDS> - GUIDE COMMAND















































      ========== COORDS COMMANDS ==========
<CORE COMMANDS>

/coords guide: guide player to saved coordinate or
inputted xyz location
USAGE: /coords guide <saved name, list path, or x y z coordinate>
EXAMPLES:
/coords guide home
/coords guide 8453 57 -4324
/coords guide 8453 -4324
/coords guide bases/TurtleMasterPDX
--
/coords add: add coordinate to personal list
USAGE
--
/coords remove: remove coordinate from personal list
--
/coords send
